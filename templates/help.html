Welcome to the Restriction Enzyme Database Help Page!

This database contains data on 5,928 different restriction enzymes. This data has been collected from the website Rebase. There are eight different search criteria that can be searched. They are:

  • Enzyme name
  • Prototype
  • Microorganism
  • Source of microorganism
  • Recognition sequence with cleavage site
  • Methylation site and type
  • Commercial source
  • Reference number

Enzyme name: Restriction enzyme name. Searching via this option will return all available data on the enzyme searched.

Prototype: The name of the first enzyme discovered with this specificity. Searching via this option will return all enzyme names that the search query is a prototype of.

Microorganism: Organism from which this enzyme had been isolated. Searching via this option will generate all enzymes that have been isolated from this organism. Each enzyme will be displayed next to the microorganism to reflect the various strains of each species.

Source of microorganism: Either an individual or a National Culture Collection. Searching via this option will return all enzymes isolated by that individual or National Culture Collection.

Recognition sequence with cleavage site: DNA sequence that the enzyme uses to determine where to cut. The ^ denotes the precise location of the cut. Searching this option will match to all enzymes that have this cut sequence. REBASE Recognition sequences representations use the standard abbreviations (Eur. J. Biochem. 150: 1-5, 1985) to represent ambiguity.

  • R = G or A
  • Y = C or T
  • M = A or C
  • K = G or T
  • S = G or C
  • W = A or T
  • B = not A (C or G or T)
  • D = not C (A or G or T)
  • H = not G (A or C or T)
  • V = not T (A or C or G)
  • N = A or C or G or T

Methylation site and type: The site of methylation by the cognate methylase when known is indicated X(Y) or X,X2(Y,Y2), where X is the base within the recognition sequence that is modified. A negative number indicates the complementary strand, numbered from the 5' base of that strand, and Y is the specific type of methylation involved:

  • (6) = N6-methyladenosine
  • (5) = 5-methylcytosine
  • (4) = N4-methylcytosine
If the methylation information is different for the 3' strand, X2 and Y2 are given as well. Searching via this option will produce all enzymes with this methylation site and type.

Commercial source: Commercial vendors who provide the isolated enzyme for use in experiments. Searching via this option will produce all enzymes available from this vendor. The vendors are searched by single letter codes, as that is how they are listed for each enzyme in the database. A chart of the vendor one letter codes corresponding to their full name is available by hovering over the relevant text on the search page.

Reference numbers: The primary reference numbers for the isolation and/or purification of the restriction enzyme or methylase, the determination of the recognition sequence and cleavage site or the methylation specificity. Searching a number will pull the matched full reference. These numbers can be obtained by first searching by enzyme name. This will list all reference numbers associated with that enzyme, which can then be used to search for the full reference.
